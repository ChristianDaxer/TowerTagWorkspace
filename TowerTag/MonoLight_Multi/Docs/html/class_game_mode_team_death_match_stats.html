Set the number of Pillars captured by a Team.

To set the overall number of Pillars in the current scene (independent of team occupancy) use -1 as teamID.

Parameters
teamIDTeamID of the Team you wan't to set the number captured Pillars for. (-1: if you wan't to set the overall count of Pillars in the scene, 0 or 1: set the number of Pillars captured by Team Fire or team Ice)
numberOfPillarsNumber of Pillars captured by the Team associated with the given teamID (if teamID is -1, the number of Pillars is saved in numberOfPillarsInScene)
