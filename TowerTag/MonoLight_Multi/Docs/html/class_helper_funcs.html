static float HelperFuncs.GetTimeDifferenceInSecFromPhotonServerTimestamps ( int  oldPhotonServerTimeStamp,
int  newPhotonServerTimestamp 
)
static

Calculates the timespan (in seconds) between to PhotonServer timestamps.

(Photons Server timestamps are in a range of [int.minValue..int.maxValue] and jump from max int value (+) to min int value (-), it starts at an arbitrary value -> so never use the absolute values, only use timespans!!!)

Parameters
oldPhotonServerTimeStampThe older Server timestamp.
newPhotonServerTimestampThe newer Server timestamp.
Returns
Timespan between old and new Server timestamp in seconds (can be negative if older > newer, but ensures right values if the timestamps jumps around from positive to negative by int overflow).
