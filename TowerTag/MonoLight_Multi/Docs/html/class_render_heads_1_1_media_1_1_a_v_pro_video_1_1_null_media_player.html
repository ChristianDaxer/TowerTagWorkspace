◆ GetTexture()

override Texture RenderHeads.Media.AVProVideo.NullMediaPlayer.GetTexture ( int  index)
virtual

Returns the Unity texture containing the current frame image.

The texture pointer will return null while the video is loading This texture usually remains the same for the duration of the video. There are cases when this texture can change, for instance: if the graphics device is recreated, a new video is loaded, or if an adaptive stream (eg HLS) is used and it switches video streams.
