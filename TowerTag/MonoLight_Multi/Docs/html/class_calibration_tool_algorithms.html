Calculates full Angle [0..360] of a direction in UnitCircle in X/Z-Plane (f^-1(angle) = direction).

Vector(1,0,0) means 0°, (1,0,1) -> 45°, (0,0,1) -> 90°, (-1, 0, -1) -> 315° and so on
