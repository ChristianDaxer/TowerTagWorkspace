Detailed Description

Attention !!! -> the UI Functions allocate Memory per Frame (OnGUI: 4 kB, UpdateUI(Simple): 1.7 kB or UpdateUI(Advanced): 0.8 kB) so the GC will collect in some time Attention !!! -> the FPS Counter should not run to long time spans, so reset them in intervals, because the Average functions for FPS/FrameTime have some shortcomings:

  • the Analytic variants can overflow because the values are added up over the whole timespan
  • the Iterative variants can accumulate calculation errors (so the deviation grows over time)
