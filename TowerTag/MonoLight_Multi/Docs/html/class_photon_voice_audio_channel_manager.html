Detailed Description

Singleton Manager class to handle unique VoiceChannelIDs for PhotonVoiceChat.

(with htese channels we try to handle all combinations of Player vs Player and Player vs Admin chat). Attention: The internal Member lookup structures need to get synced from Master to all clients (use the unified Serialize function with write/read BitSerializer streams to Serialize/Deserialize)
