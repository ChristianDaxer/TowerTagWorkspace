Detailed Description

Different channels voices events are sent to diferent enet channels.

Single event code is enough for all voice communications. Because of EventData does not provides ChannelID, we are forced to encode channel id in event code. For this purpose, range of event codes of PhotonPeer.ChannelCount length starting from Code0 used.
