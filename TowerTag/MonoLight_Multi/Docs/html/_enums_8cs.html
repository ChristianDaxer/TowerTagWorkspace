◆ EncryptionMode

enum EncryptionMode
strong

Defines how the communication gets encrypted.

Enumerator
PayloadEncryption 

This is the default encryption mode: Messages get encrypted only on demand (when you send operations with the "encrypt" parameter set to true).

DatagramEncryption 

With this encryption mode for UDP, the connection gets setup and all further datagrams get encrypted almost entirely.

On-demand message encryption (like in PayloadEncryption) is skipped.

This mode requires AuthOnce or AuthOnceWss as AuthMode!
