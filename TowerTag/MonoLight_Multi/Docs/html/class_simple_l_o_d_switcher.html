Detailed Description

Class to realize own LOD-System to switch whole GameObject hierarchies as LODLevels (so we can enable/disable Lights/Particle Effects etc.

and not only Renderer like Unitys build in LOD System). One of these should be in every scene in which we want to use this system. Uses custom SimpleLODGroup class. If no LODGroups set in lodgroups array we search for all available LODGroups in scene and register them (so be sure you set them right, if you have more than one SimpleLODSwitcher Component in a scene, which should be avoided if not neccesary).
