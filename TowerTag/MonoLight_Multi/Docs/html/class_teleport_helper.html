Convinience function to ease respawns.

It teleports player to a spwanPillar (if possible), resets Players health, sets PlayerState (to immortal, gunDisabled, isInLimbo) & reactivates the PlayerState after given reactivationTimeout. Call only on Master! Don't use it for teleports requested by clients!

Parameters
playerToSpawnThe Player who should get spawned.
spawnPillarThe Pillar we wan't to spawn on.
teleportDurationTypeType of Teleportation (needed to calculate teleport duration).
reactivationTimeoutTimeout after which PlayerState gets reactivated again.
